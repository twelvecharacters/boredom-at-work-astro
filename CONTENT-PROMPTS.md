# Content Prompts for Boredom at Work

## Article Generator Prompt

```
Write a blog article for boredom-at-work.com about [TOPIC].

Target audience: Office workers who are bored at work and want to research/learn about interesting topics during downtime.

Requirements:
- Title: Catchy, SEO-friendly (include main keyword)
- Length: 1500-2500 words
- Tone: Casual, friendly, informative (like explaining to a coworker)
- Language: English

Structure:
1. Hook intro (why this is interesting to research at work)
2. What is [TOPIC]? (beginner-friendly explanation)
3. Main content (practical info, comparisons, tips)
4. Pros and cons (if applicable)
5. Conclusion with actionable next steps

Include:
- Practical tips someone can use
- Price ranges (if products involved)
- Comparison tables (if comparing options)
- "Things to research further" section

SEO elements needed:
- Meta description (150-160 characters)
- 3-5 relevant tags
- FAQ section (3 questions with answers)

Format the output as Astro markdown with this frontmatter:
---
title: ""
description: ""
publishDate: YYYY-MM-DD
author: "bored chap"
image: ""
tags: []
faq:
  - question: ""
    answer: ""
---
```

## Quick Article Ideas Prompt

```
Generate 10 blog article ideas for boredom-at-work.com about [CATEGORY].

The site is for office workers who want to productively use their downtime.

For each idea provide:
1. Article title (SEO-friendly)
2. Target keyword
3. Search intent (informational/commercial/transactional)
4. Estimated competition (low/medium/high)
5. One-sentence summary

Categories to consider:
- Hobbies (3D printing, photography, etc.)
- Shopping guides (Japan imports, niche products)
- Tech & gadgets
- Skills & learning
- Productivity
```

## Product Comparison Prompt

```
Write a comparison article: "[PRODUCT A] vs [PRODUCT B]" for boredom-at-work.com.

Context: Someone bored at work researching which to buy.

Include:
1. Quick verdict (TL;DR at the top)
2. Comparison table (features, price, pros/cons)
3. Detailed breakdown of each
4. Who should buy which
5. Where to buy (with affiliate potential in mind)
6. FAQ section

Tone: Helpful friend giving honest advice, not salesy.
```

## Beginner Guide Prompt

```
Write a beginner's guide: "[TOPIC] for Complete Beginners" for boredom-at-work.com.

Assume the reader knows nothing about [TOPIC] and is researching during work downtime.

Structure:
1. What is [TOPIC] and why it's interesting
2. Key terminology explained simply
3. How to get started (step by step)
4. Common beginner mistakes to avoid
5. Recommended first purchases/resources (budget-friendly)
6. What to learn/research next
7. FAQ for beginners

Include a "Quick Start Checklist" box.
```

## Japan Shopping Guide Prompt

```
Write a guide about buying [PRODUCT/CATEGORY] from Japan for boredom-at-work.com.

Include:
1. Why buy from Japan (exclusive items, prices, quality)
2. Best proxy services comparison (Buyee, Sendico, etc.)
3. Step-by-step buying process
4. Shipping costs and times
5. Customs/import duties to expect
6. Tips to save money
7. Common pitfalls to avoid
8. FAQ

Target reader: Someone in US/EU curious about importing from Japan.
```
